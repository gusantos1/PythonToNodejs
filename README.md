## list.append → array.push
<table>
<tr>
<td>
Python

  ```python
  lista = [1, 2 , 3]
  lista.append(4)
  print(lista)
  ```
</td>
<td>
Nodejs

  ```javascript
  const lista = [1, 2 , 3];
  lista.push(4);
  console.log(lista);
  ```
</td>
<td>
Saída

  ```javascript
  [ 1, 2, 3, 4 ]
  ```
</td>
